## Explanation
The idea is to maintain two pointers, `i` and `j`, which will initially point to the first array element. Pointer `j` will iterate through the array until it reaches `n`, and during this process, we will keep incrementing the sum by `arr[j]`. If the sum becomes greater than $x$, we will keep decrementing the sum by `arr[i]` and keep incrementing `i` until the sum becomes less than or equal to $x$. Finally, we will increment our result if the sum becomes equal to $x$.

## Implementation
**Time complexity:** $\mathcal{O} (n)$

<CPPSection>
```cpp
#include <bits/stdc++.h>
#define int long long

int32_t main() {
	int n, x;
	std::cin >> n >> x;
	std::vector<int> arr(n);
	for (int i = 0; i < n; i++) { std::cin >> arr[i]; }

	int i = 0, j = 0;
	int res = 0, sum = 0;
	while (j < n) {
		sum += arr[j];
		while (sum > x) {
			sum -= arr[i];
			i++;
		}
		res += (sum == x);
		j++;
	}
	std::cout << res << '\n';
	return 0;
}
```
